### CR 2: Calculate the total revenue generated by all rentals in the store.


```
Computational requirement: Calculate the total revenue generated by all rentals in the store.


+ Test Case 1: "Single Rental Calculation" 
    Input: Calculate total revenue from rentals with multiple rental records.
    SetUp:  
    1. Create a store instance.
    2. Add Rental object with car plate "ABC123", model "Toyota Camry", daily price: 100 CNY, rented for 3 days with back date "2025-11-12", due date "2025-11-13", rental date "2025-11-10".
    3. Add Rental object with car plate "XYZ789", model "Honda Civic", daily price: 150 CNY, rented for 2 days with back date "2025-11-11", due date "2025-11-13", rental date "2025-11-10".
    4. Add Rental object with car plate "LMN456", model "Ford Focus", daily price: 200 CNY, rented for 1 day with back date "2025-11-12", due date "2025-11-13", rental date "2025-11-12".
    5. Calculate the total revenue.
    Expected Output: Total revenue = (100 * 3) + (150 * 2) + (200 * 1) = 300 + 300 + 200 = 800 CNY
---
+ Test Case 2: "No Rentals Calculation" 
    Input: Calculate total revenue from rentals with no rental records.
    SetUp:  
    1. Create a store instance.
    2. Verify there are no Rental objects added.
    3. Calculate the total revenue.
    Expected Output: Total revenue = 0 CNY
---
+ Test Case 3: "Multiple Rentals with Same Daily Price" 
    Input: Calculate total revenue from rentals where multiple cars have the same daily price.
    SetUp:
    1. Create a store instance.
    2. Add Rental object with car plate "CAR001", model "Chevrolet Malibu", daily price: 120 CNY, rented for 2 days with back date "2025-11-13", due date "2025-11-13", rental date "2025-11-12".

+ Test Case 4: "Mixed Prices Calculation"

    4. Calculate the total revenue.
    Expected Output: Total revenue = (180 * 1) + (220 * 1) = 180 + 220 = 400 CNY
```
